Modificar Punto

  • Este campo te permite ingresar un nombre para el punto que será visible para acceder a él. Puedes proporcionar un nombre descriptivo que ayude a identificar el contenido del punto. Por ejemplo, podrías utilizar un nombre que refleje el propósito del mismo. Asegúrate de elegir un nombre claro y significativo para facilitar su identificación y acceso por parte de los usuarios.

Este campo es requerido.
El campo debe tener al menos 5 caracteres.
El campo no puede exceder los 100 caracteres.
El campo no puede contener caracteres no permitidos.
